📱 Beauty Den - Customer API Documentation

🔒 Encryption Requirement:
All responses are encrypted using AES-128-CBC. You will receive a string: "ENCRYPTED_DATA:IV". You must decrypt this using your shared key to get the JSON result.

1. Send OTP (Login / Register)

Initiates OTP process for existing login or new user registration.

POST /V1/customer/sendOtpOnMobileNumber
Field Type Presence Description
mobile_number String Required International format (+91...)
name String New User Full name of the customer
city_id Integer New User Numeric ID of the city

2. Verify OTP

Validates OTP and returns authentication token.

POST /V1/customer/verifyOtpOnMobileNumber
Field Type Presence Description
mobile_number String Required Same as above
otp String Required 6-digit numeric string

3. Update FCM Token

Updates the Firebase token for the user. Call this whenever the user opens the Home Screen.

POST /V1/customer/updateFcmToken
Bearer Auth Required
Field Type Presence Description
user_id Integer Required Numeric ID of logged-in user
fcm_token String Required Token from Firebase
device_type String Optional "android" or "ios"
device_name String Optional Model name (e.g., iPhone X)

4. Get Cities List

Provides a list of all cities to choose from.

GET /V1/cities

Returns an array of city objects: [{"id": 1, "name": "Surat"}, ...]
