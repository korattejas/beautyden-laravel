Browse & Search Data (Search Home)

Universal endpoint for category browsing, sub-category exploration, and free-text service searching. Implements a 3-step drill-down: Category → Sub-Category → Services.

POST /V1/customer/getServiceSearchData Token Required
Field Type Description
city_id Integer Required. Prices/Availability will be based on this city.
search String Optional. Search services by name/description.
category_id Integer Optional. Set this to get Sub-Categories.
sub_category_id Integer Optional. Set this to get Services.
Business Logic:
• Initial Load: If only city_id is sent, it returns all top-level Categories.
• City Check: If city is "Coming Soon" (status != 0), it returns is_coming_soon: true.
• Navigation: Every response includes the categories array to allow quick switching/filtering in UI.
Response Example (Initial View)
{
    "code": 200,
    "status": true,
    "message": "Data retrieved successfully.",
    "data": {
        "categories": [ ... all categories for navigation bar ... ],
        "type": "categories",
        "results": [ ... all active categories to display in grid ... ]
    }
}
Response Example (Category Clicked)
{
    "code": 200,
    "status": true,
    "message": "Data retrieved successfully.",
    "data": {
        "categories": [ ... ],
        "type": "subcategories",
        "results": [ ... list of sub-categories for the selected category ... ]
    }
}

POST /V1/customer/submitReview Token Required
Field Type Description

review String Required. Customer's feedback text.
photos[] File[] Optional. Multiple images (jpeg, png, jpg, webp, max 2MB each).

13.1 Send Login OTP / Registration Check

Checks if the beautician exists and sends OTP. If not found, informs the app to show the registration form.

POST /V1/beautician/sendLoginOtp
Field Type Description
mobile_number String Required. Beautician's mobile number.
name String Required only for Registration.
dob Date Required only for Registration. (YYYY-MM-DD)
bio String Required only for Registration.
experience_years Numeric Required only for Registration.
address String Required only for Registration.
Flutter Flow:
1. Call this API with only mobile_number.
2. If is_exists: false, show registration form and call this API again with all fields.
3. If is_exists: true, move to OTP verification screen.
