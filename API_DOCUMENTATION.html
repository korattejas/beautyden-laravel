1. Authentication Flow

Send OTP

Initiates OTP delivery via MSG91 WhatsApp/SMS.

POST /V1/customer/sendOtpOnMobileNumber
Field Type Description

5. Address Management

Save/Update Address

Saves a new address or updates an existing one (up to 3 addresses). If marked as default, it updates the main user profile.

POST /V1/customer/saveUserAddress Token Required
Field Type Description
id Integer Optional. Address ID (to update existing)
address String Full address string
city_id Integer ID of the city
latitude Numeric Optional
longitude Numeric Optional
type String Optional (Home, Work, Other)
is_default Boolean If true, updates main profile city/address
Exact Response
{
    "code": 200,
    "status": true,
    "message": "Address saved successfully.",
    "data": {
        "id": 1,
        "user_id": 4,
        "address": "123, Street Name, Ahmedabad",
        "latitude": "23.0225",
        "longitude": "72.5714",
        "type": "Home",
        "is_default": true,
        "created_at": "2026-04-24T05:00:00.000000Z",
        "updated_at": "2026-04-24T05:00:00.000000Z"
    }
}
